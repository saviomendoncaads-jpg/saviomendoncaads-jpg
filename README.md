## Projetos

### [Ultron Assistant](https://github.com/saviomendoncaads-jpg/ultron-assistant)
Assistente de voz pessoal com IA | Python | Groq | PyQt6

Assistente de voz com interface HUD sci-fi, controle total do Windows por voz,
memoria baseada no vault do Obsidian e personalidade sofisticada.

Destaques tecnicos:
- Interface HUD animada a 60fps com PyQt6 e QPainter
- LLM via Groq llama-3.3-70b com fallback automatico entre 4 modelos
- RAG local com BM25 indexando 13 mil notas do Obsidian
- Classificador de intencao hibrido com regex e LLM 8b
- Mais de 25 ferramentas: apps, mouse, teclado, browsers, WhatsApp, Gmail, YouTube
- TTS neural gratuito com edge-tts em pt-BR

Tecnologias: Python | PyQt6 | Groq API | BM25 | Playwright | PyAutoGUI | edge-tts
